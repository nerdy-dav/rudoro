# Rudoro

A TUI Pomodoro timer written in Rust.

## Usage

```bash
rudoro
```

Simple as that.

This will give you the default work/rest timing of 25 minutes work and 5 minutes rest.

You can change this:

```bash
rudoro -w 20 -r 10
```

For 20 minutes work and 10 minutes rest...

-r = rest time in minutes
-w = work time in minutes
